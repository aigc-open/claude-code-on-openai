🛠️ 使用 claude-code 工具

claude-code 是 Anthropic 官方提供的命令行工具
通过本代理服务，可以使用各种 OpenAI 兼容的大模型

📦

安装 claude-code 工具

使用 npm 全局安装 claude-code 命令行工具
确保您的系统已安装 Node.js 环境

npm install -g @anthropic-ai/claude-code

安装完成后，您可以在任何目录下使用 claude 命令

⚙️

配置环境变量

设置代理服务地址和 API 密钥，支持多种模型选择

基础 URL
ANTHROPIC_BASE_URL=http://ph8.co/claude_code/gpt-4o
API 密钥
ANTHROPIC_API_KEY=your-api-key
💡 支持的模型
您可以将 URL 中的模型名称替换为以下任意模型：
# GPT 系列 http://ph8.co/claude_code/gpt-4o http://ph8.co/claude_code/gpt-3.5-turbo # 国产模型 http://ph8.co/claude_code/qwen2.5-7B http://ph8.co/claude_code/deepseek-v3
🚀

启动 claude-code

一行命令启动 claude-code，开始您的 AI 编程之旅

🎯 快速启动
复制以下命令，替换您的 API 密钥后执行：
ANTHROPIC_BASE_URL=http://ph8.co/claude_code/gpt-4o ANTHROPIC_API_KEY=your-api-key claude
💻 使用示例
启动后，您可以直接与 AI 对话，进行代码编写、调试、优化等操作：
$ claude Claude: 您好！我是 Claude，您的 AI 编程助手。 我可以帮您编写代码、解决问题、优化算法等。 请告诉我您需要什么帮助？ You: 帮我写一个 Python 快速排序算法 Claude: 当然！我来为您编写一个 Python 快速排序算法...
